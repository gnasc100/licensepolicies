Waiver Wizard Privacy Policy

All the data collected by this app is stored within your device. The Waiver Wizard Android Application does not send or store data in any servers. The option " Report a Bug -> Send Report" allows you to send the Applications internal log files via e-mail to for troubleshooting purposes only. This option can only be triggered manually by yourself. The data in the log file consists of internal error messages and the screens that were opened. No personal data is included in this log.

This app contains advertisement from Google Ads(TM by Google). For Google privacy policy please refer to: https://policies.google.com/privacy?hl=en .
